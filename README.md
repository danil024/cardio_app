# Cardio HR Monitor

Приложение для мониторинга пульса в реальном времени с датчиком GEOID HS500.

## Возможности

- Подключение к пульсометру GEOID HS500 по Bluetooth (BLE)
- Отображение пульса крупными цифрами
- График пульса за последние 3 минуты
- Расчёт зон пульса по возрасту (формула 220 - возраст)
- Звуковые оповещения при выходе из целевой зоны (снизьте/увеличьте темп)
- Сигнал опасности при превышении 90% макс. пульса
- Сохранение сессий в CSV и JSON

## Запуск

```bash
flutter pub get
flutter run
```

При первом запуске выполните `flutter create .` для генерации недостающих файлов (иконки, iOS конфигурация), если проект был создан вручную.

## Требования

- Flutter SDK >= 3.2.0
- Android 5.0+ (API 21)
- iOS 12.0+ (для будущей поддержки)

## Структура

- `lib/data/ble/` — BLE сервис для пульсометра
- `lib/data/storage/` — сохранение сессий и настроек
- `lib/data/audio/` — звуковые оповещения
- `lib/domain/` — модели и расчёт зон
- `lib/presentation/` — UI экраны
