# Changelog

All notable changes to this project are documented in this file.

## v1.0.0 - Initial public release

- Implemented BLE heart-rate workflow with GEOID HS500 focused discovery.
- Added real-time HR chart with dynamic vertical range and detailed grid.
- Added large BPM overlay on chart.
- Implemented five-zone heart-rate model with visual zone indication.
- Added alert engine with sound and TTS notifications.
- Added timer/stopwatch module with roller-based time picker.
- Added media control module with system Android media key integration.
- Added workout history with CSV/JSON autosave and delete actions.
- Added EN/RU localization with linked interface + voice language switch.
- Added downloadable release artifacts to `releases/`:
  - APK
  - AAB
  - iOS source archive for macOS/Xcode build.
