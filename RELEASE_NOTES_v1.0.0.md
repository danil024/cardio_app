# Cardio Pulse v1.0.0

## Highlights

- BLE heart-rate monitoring flow for GEOID HS500 and compatible devices.
- Real-time chart + large BPM overlay UI.
- Five-zone HR model with visual and audio/voice guidance.
- Timer/stopwatch + media controls in unified compact module style.
- Session history with automatic CSV/JSON logging.

## Downloads

- APK: `releases/CardioPulse-v1.0.0-release.apk`
- AAB: `releases/CardioPulse-v1.0.0-release.aab`
- iOS source package: `releases/CardioPulse-iOS-source-v1.0.0.zip`

## iOS note

`.ipa` build is not generated on Linux.  
Use macOS + Xcode to build/sign iOS binaries from the included source package.
